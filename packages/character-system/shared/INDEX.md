# Character System Protocol Index

This directory is the package-local protocol layer for character generation,
runtime use, diagnosis, maintenance, and generalization.

- `character_skill_spec.md`: character skill layout and responsibilities.
- `drift_taxonomy.md`: character-output drift vocabulary.
- `handoff_format.md`: diagnosis-to-maintenance handoff shape.
- `patch_protocol.md`: diagnosis and maintenance authority boundary.
- `runtime_loop_policy.md`: durable runtime correction workflow.
- `future_drift_policy.md`: package drift prevention.
- `versioning_policy.md`: character-system version records.
- `protocol_manifest.json`: machine-readable package protocol registry.
- `templates/`: runtime-loop record templates.
- `schemas/`: lightweight runtime-loop and protocol schemas.

Workspace-global governance remains under the root `shared/` directory.
