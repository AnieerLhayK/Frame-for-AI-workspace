# Todo Index

This directory tracks unfinished or condition-triggered work without mixing
workspace optimization with external skill intake.

- [`workspace-optimization.md`](workspace-optimization.md): workspace-native
  architecture, governance, prompt-library, character-system, and deferred
  optimization work.
- [`external-skills.md`](external-skills.md): every raw skill source discovered
  under `workspace_manifest.yaml -> external_roots.raw_skills`, plus its
  provenance, compatibility, adaptation, validation, and registration status.

Always-on operating rules belong in `AGENTS.md`,
`PROJECT_CONTEXT/context_budget.md`, and shared policies rather than being
repeated as tasks here. A new raw skill must be added to
`external-skills.md` before it is evaluated or adapted.
