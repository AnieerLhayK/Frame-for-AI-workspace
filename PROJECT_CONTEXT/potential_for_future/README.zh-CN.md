# Potential For Future

此目录保存需要长期记住、但尚未成为当前 todo 或强制策略的规划材料。

## 分类

- `optimization_options.yaml`：潜在的工作区、架构、工具或工作流优化项。
- `risk_register.yaml`：潜在的结构性风险、暴露风险和迁移风险，在触发条件出现时重新检查。

这里的条目是检索入口，不是实施计划。只有在当前证据和范围得到确认后，才应将条目提升为任务路由、策略、活跃 todo 或源代码变更。

新增分类时，应同步更新本说明、英文 companion、`PROJECT_CONTEXT/knowledge_registry.yaml` 以及需要发现它的任务路由规则。历史任务台账不会因为注册表移动而重写。
