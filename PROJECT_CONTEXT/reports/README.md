# PROJECT_CONTEXT Reports

This directory stores human-oriented context summaries, session summaries, and architecture notes.

It does not replace workspace `reports/`.

Use this directory for:

- phase summaries;
- session summaries;
- architecture notes;
- long-term continuity notes.

Do not store machine-readable validation outputs here. Those belong in `reports/` or `packages/character-system/reports/runtime-loop/` depending on the workflow.
