# Reports

Use this directory for knowledge-base reports, including:

- engineering retrospectives;
- architecture reviews;
- skill evolution summaries;
- governance audits.
- knowledge-layer publication and provenance reviews.

This does not replace workspace-level `reports/`. Workspace reports describe this repository's state. These reports should capture reusable engineering lessons.

The existing Skill Engineering setup report is retained as a historical
snapshot from before the knowledge layer was promoted.
