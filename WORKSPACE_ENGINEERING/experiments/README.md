# Experiments

Use this directory for exploratory engineering notes, including:

- structures that were tried but not adopted;
- abandoned designs;
- exploratory runtime ideas;
- portability experiments;
- validator experiments.

Experiments should clearly say whether they are active, rejected, deferred, or only speculative.

Record the hypothesis, bounded test surface, expected evidence, actual result,
rollback, and promotion decision. Externally inspired experiments must retain
their provenance.
