# Templates

Use this directory for reusable engineering templates, such as:

- prompt templates;
- governance templates;
- validator templates;
- runtime packet templates;
- future reusable skill patterns.
- case-study and external-knowledge intake structures.

This directory is for knowledge-base templates. Runtime loop packet templates used by the current workspace belong in `packages/character-system/shared/templates/`.

Templates should encourage attribution, evidence levels, verification,
limitations, privacy review, and rollback rather than presenting advice as
universal truth.
