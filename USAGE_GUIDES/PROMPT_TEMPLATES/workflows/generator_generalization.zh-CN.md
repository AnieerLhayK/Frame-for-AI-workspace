# 工作流 Prompt：生成器泛化

仅在维护者审查后且在声明的曝光平台上与 `character-generator` 配合使用。

```text
审查此维护者批准的泛化候选。

泛化说明：
<路径或粘贴的说明>

关联 patch：
<patch id 或路径>

源角色：
<角色 id>

任务：
决定此经验是否应影响 character-generator 资产。

请：
1. 检查该经验是否确实可泛化。
2. 如果接受，识别最小的生成器影响面。
3. 如果该经验是角色特定的或证据不足，则拒绝或推迟。
4. 除非我明确批准此实施轮次，否则不要编辑生成器文件。

决定选项：
- accepted
- rejected
- deferred

输出：
- 决定
- 原因
- 如果接受，目标生成器文件
- 如果泛化的风险
- 验证计划
```
